Преобразователь кастомных имен в локализацию с последующей сортировкой на переведённые и не переведённые строки
Производится в несколько этапов, т.к. есть ограничения по регулярным выражениям (либо я сделал кое-что криво)
Made by Assin

################################################################################################
Для преобразования формы правления строчными буквами

	1) Найти:
	text = {.*\r\n		localisation_key = (.*)_(IM|RD|DT|VN|TV|PR)(\r\n.*){0,5}\r\n	}

	Заменить на:
 \1_\2:0 "\1" #GetGovernment\2

  	2) Найти:
	text = {.*\r\n		localisation_key = (.*)_(IM|RD|DT|VN|TV|PR)(\r\n.*){5,10}\r\n	}

	Заменить на:
 \1_\2:0 "\1" #GetGovernment\2

   	3) Найти:
	text = {.*\r\n		localisation_key = (.*)_(IM|RD|DT|VN|TV|PR)(\r\n.*){10,20}\r\n	}

	Заменить на:
 \1_\2:0 "\1" #GetGovernment\2

	4) Найти:
defined_text = {\r\n	name = GetGovernment(.*)\r\n

	Заменить на: [Пустую строку]

	5) Найти:
^	#.*

	Заменить на: [Пустую строку]
 
	6) Удалить лишние строки в начале файла, удалить все "}"
	7) Правка -> Операции со строками -> удалить дублирующиеся строки
	8) В переведённом файле:
	Найти:
(.*):0(.*)

	Заменить на:
\1:1\2

	9) Добавляем переведённые русские строки во временный файл
	10) Сортировка по возрастанию, убрать всё лишнее
	11) Найти:
(.*):0(.*)\r\n\1:1(.*)

	Заменить на:
\1:1\3
